TimeML Annotation

Use the mouse to highlight text, then press 'Tag Event' to tag that text as an event, or press 'Tag Time' to tag that text as a time.

Keyboard shortcuts:

  • E: Tag Event
  • T: Tag Time
  • U: Undo
  • ?: Toggle Help
